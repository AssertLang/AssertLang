AssertLang Networking Flow
1. Prompt (User / Agent)
⬇️
2. Runner (Python / Node / Go)
Executes generated code
⬇️
3. Internal Bind:
UDS Socket (/run/mcpd/sock/<task_id>.sock)
(Fallback: high-range TCP 61000–64999)
⬇️
4. AssertLang Gateway
Listens on Port 23456
Stable, public-facing
⬇️
5. User Access:
http://127.0.0.1:23456/apps/<task_id>/
